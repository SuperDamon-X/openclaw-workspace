# OpenClaw paths (Windows)

- State dir: `C:\Users\Administrator\.openclaw`
- Config: `C:\Users\Administrator\.openclaw\openclaw.json`
- Workspace git repo: `C:\Users\Administrator\.openclaw\workspace`
- Gateway task: `OpenClaw Gateway (Interactive)`
- Watchdog script: `C:\Users\Administrator\.openclaw\gateway_watchdog.ps1`
